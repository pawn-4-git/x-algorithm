X (旧Twitter) のアルゴリズム徹底解説

Grokベースの最新AIが、ユーザーの過去の行動履歴を元に「どのくらいその投稿に反応（エンゲージメント）しそうか」を予測し、その確率に重み付け（加点・減点）をしてタイムラインの表示順位を決めています。

❤️ プラス評価（スコアが上がる要因）

AIが「このユーザーはこの投稿に対して以下のアクションを起こす確率が高い」と予測した場合、それぞれの確率に重み（ウェイト）が掛けられて総合スコアが加点されます。
予測スコア = (確率 × ウェイト) の総和
例: 総合スコア = (いいね確率 × 0.5) + (リポスト確率 × 20.0) + (リプライ確率 × 27.0) + ...
※ 具体的なウェイト数値はハードコードされておらず、xai_feature_switches を通じて外部から動的に読み込まれます（🔗ウェイト数値の取得元ソース）。記載している数値は、過去の公開時に判明した目安となる設定値です。
  • 👍 いいね (Favorite) 🔗ソース 投稿に「いいね」を押す確率。アルゴリズムにおける最も基本的な強力なプラス評価です。
    ウェイト: 0.5倍
  • 💬 返信 (Reply) 🔗ソース 投稿に対してリプライ（返信）を送る確率。会話を生む投稿として高く評価されます。
    ウェイト: 27.0倍投稿者の返信あり: 75.0倍
  • 🔁 リポスト (Retweet / Quote) 🔗ソース リポストや引用リポストで拡散する確率。情報が広がる可能性として加点されます。
    ウェイト: 20.0倍
  • 👤 プロフィールクリック 🔗ソース 投稿者のプロフィールページへ遷移する確率。投稿者への興味の表れとみなされます。
    ウェイト: 12.0倍
  • ➕ 投稿者のフォロー 🔗ソース その投稿を見たことがきっかけで、新しく投稿者をフォローする確率。非常に高い価値があります。
    ウェイト: 4.0倍 (推定)
  • 👀 滞在時間 (Dwell) 🔗ソース スクロールを止めてじっくりと投稿を読んだ時間、またはクリック後の継続的な滞在時間。
    ウェイト: 10.0倍 (推定)
  • 🖼️ 画像拡大・動画視聴 🔗ソース 画像をタップして拡大する確率（Photo Expand）や、動画を一定時間視聴する確率（VQV）。
    動画視聴: 0.005倍
  • 🔗 シェア (DM・リンク等) 🔗ソース 投稿をDMで送る、またはリンクをコピーして外部へシェアする確率。
  • 🖱️ リンククリック等 🔗ソース 投稿内のリンクや、引用された投稿をクリックする確率（Quoted Click / Click）。

📉 マイナス評価（スコアが下がる要因）

逆に、ユーザーが不快に感じたり、興味を持たないと予測される行動は、総合スコアから大きく減点されます。マイナス要因のウェイトは総じて重く設定されています。
ソースコード内の offset_score 関数にて、最終スコアがマイナスになった場合はペナルティのオフセット計算が行われます。
総合スコア < 0 の場合:
最終スコア = (総合スコア + ネガティブ総和) / 全ウェイト総和 * ペナルティオフセット
※ マイナスのペナルティ数値も同様に動的に読み込まれます（🔗ネガティブ総和の計算ソース、🔗ペナルティオフセット関数のソース）。
  • 🥱 興味なし (Not Interested) 🔗ソース ユーザーが「この投稿に興味がない」メニューを押す確率。
    ペナルティ大 (-74.0倍など)
  • 🚫 ブロック (Block Author) 🔗ソース 投稿を見た直後に、その投稿者をブロックする確率。
    ペナルティ特大 (-74.0倍など)
  • 🔇 ミュート (Mute Author) 🔗ソース 投稿を見た直後に、その投稿者をミュートする確率。
    ペナルティ特大 (-74.0倍など)
  • 🚨 通報 (Report) 🔗ソース スパムや違反行為として通報（レポート）される確率。
    ペナルティ特大 (-369.0倍など)
  • ⏭️ 即スクロール (Not Dwelled) 🔗ソース タイムラインに表示されたものの、全く読まれずに一瞬でスクロールして読み飛ばされる確率。
    魅力なし判定

⚠️ 限定度合い・スコア補正（ペナルティ＆ブースト）

予測された行動確率の合計スコアに対して、さらにタイムラインの多様性やフォロー外の調整が行われます。
  • 👤 同一投稿者のペナルティ 🔗ソース タイムラインで同じ人の投稿ばかりが続くのを防ぐ機能です。同じ投稿者のツイートがフィードに複数回出現するたびに、乗数（減衰率）が掛けられ、徐々にスコアが低下します。
    diversity_multiplier = (1.0 - floor) * (decay_factor ^ position) + floor
    最終スコア = 元スコア × diversity_multiplier
  • 🌐 フォロー外の補正 🔗ソース 自分がフォローしていない人（Out-Of-Network）の投稿には、特別なウェイトが掛けられます。通常はスコアが調整されますが、トピックに一致する場合などは柔軟に変動します。
  • 🐣 新規ユーザー向けブースト 🔗ソース アカウントを作ったばかりでフォロー数が少ない新規ユーザーに対しては、フォロー外の優れたコンテンツがより多く表示されるように特別なブースト（加点乗数）がかかります。

🚫 完全に除外されるフィルター (Visibility Filters)

スコアの高さに関わらず、以下のいずれかに該当する投稿はタイムラインに表示される前に自動で弾かれます。
  • 👁️ 既読済み・表示済み 🔗ソース 過去にすでに見た投稿や、今回のセッションですでに表示された投稿。
  • 🔁 重複・連投 🔗ソース 全く同じ投稿IDの重複や、同じリポスト、同じスレッドの別ブランチの重複表示。
  • ⏳ 古すぎる投稿 🔗ソース 一定の時間が経過し、鮮度が落ちた古い投稿。
  • 🙈 自分の投稿 🔗ソース おすすめタイムライン（For You）には自分自身の投稿は表示されません。
  • 🔕 ミュート＆ブロック 🔗ソース ブロック中のユーザー、または設定したミュートキーワードが含まれる投稿。
  • 🔒 課金限定コンテンツ 🔗ソース ユーザーがサブスクリプション登録していない有料限定コンテンツ。
  • 🚨 規約違反 🔗ソース スパム、暴力、グロテスク、削除済みなど、プラットフォームの安全基準を満たさない投稿。

✨ おすすめアルゴリズムの攻略法まとめ

現在のXのアルゴリズムは、小手先のハックではなく「どれだけユーザーの感情を動かし、行動を引き起こせるか」をGrok（AI）が純粋に予測しています。

1. プラスのアクションを誘発する：いいね、リプライ、滞在時間、画像タップなど、読者が思わず手を止めてリアクションしたくなるコンテンツ作りが最重要です。
2. ネガティブアクションを避ける：過度な煽りやスパム的な投稿は、ブロックや「興味なし」を押された瞬間に致命的な減点を受けます。
3. 連投に注意：短い時間に連続して投稿すると、「多様性のペナルティ」を受けてスコアが下がるため、間隔を空けることが推奨されます。
